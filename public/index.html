✨ Importador Automático

Adicione um botão mágico aos seus favoritos! Quando você estiver no perfil de um restaurante no Instagram, clique nele para importar nome, foto, link e bairro automaticamente para cá.

➕ Importar p/ RolêPOA
Como instalar e usar no Computador:
  1. Exiba a barra de favoritos do seu navegador usando Ctrl + Shift + B (ou Cmd + Shift + B no Mac).
  2. Arraste o botão colorido acima diretamente para a sua barra de favoritos.
  3. Acesse o Instagram de qualquer restaurante de Porto Alegre no navegador.
  4. Clique no favorito "➕ Importar p/ RolêPOA" e a aba abrirá o app com todos os dados preenchidos prontos para salvar!
