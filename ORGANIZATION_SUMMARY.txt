🚀 COMO USAR AGORA
───────────────────────────────────────────────────────────────────────────────

1. INSTALAR DEPENDÊNCIAS
   $ npm install

2. RODAR SERVIDOR LOCAL (porta 5500)
   $ node config/.serve.js

3. RODAR TESTES E2E
   $ npm run test:e2e
   $ npm run test:e2e:headed    (com navegador visível)
   $ npm run test:e2e:ui        (interface interativa)

4. ABRIR NO NAVEGADOR
   http://127.0.0.1:5500
